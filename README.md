# Hanzi-Learning
learning app with memo cards to learn chinese characters from french

## How to use

### First time
just run
```
./setup.sh
```
this will setup the database and add the hsk characters before launching the app

Note: there is only up to hsk3 in `data/`

### Then
run
```
npm run dev
```
